Git Configuration Settings

Set Your Global Username

Configure your display name for all Git repositories on your system:

git config --global user.name "Your Name"

Set Your Global Email Address

Configure your email address for all Git repositories on your system:

git config --global user.email "your_email@example.com"

Set a Repository-Specific Username

Override the global username for a specific Git repository (run this command inside the repository): 

git config user.name "Your Name"

Set a Repository-Specific Email Address

Override the global username for a specific Git repository (run this command inside the repository): 

git config user.email "your_email@example.com"

View Git Configuration

View Global Username

Check your globally configured Git username:

git config --global user.name

View Global Email

Check your globally configured Git emial:

git config --global user.email

Check specific config repo name

Check Configured user name add for specific repository

git config user.name

Check specific config repo email

Check Configured user email add for specific repository

git config user.email

Initialize a Git Repository

Create a New Repository

Initialize Git in the current directory to start version tracking:

git init

Stage Files for Commit

Stage a Specific File

Add a single file to Git's staging area:

git add fileName.extension

Stage All Changes

Add all new and modified files to the staging area:

git add .

Save Changes with Commit

Create a New Commit

Permanently save your staged changes to version history with a descriptive message:

git commit -m "commit message here"

Connect Local Project to GitHub

Link Existing Folder to New Repository

Connect your local project folder to a new GitHub repository:

git remote add origin https://github.com/username/repo.git

Link Existing Folder to Existing Repository

Connect your local folder to an already created GitHub repository:

git remote add origin https://github.com/username/repo.git
git pull origin main

Upload Changes to Remote

Push to Main Branch

Upload your committed changes to the main branch of the remote repository:

git push origin main

Push Specific Branch

Upload changes from a different local branch to its remote counterpart:

git push origin branch_name

Force Push

Overwrite remote history (use with caution):

git push --force origin main

View Commit History

Basic Commit Log

Display the commit history in chronological order:

git log

Compact Log View

Show simplified commit history (one line per commit):

git log --oneline

Graphical Branch History

Visualize commit history with branch topology:

git log --graph --oneline --all

Filter by Author

Show commits from a specific contributor:

git log --author="Name"

Search Commit Messages

Find commits containing specific text:

git log --grep="search term"

Show Changes in Files

View what changed in each commit:

git log -p

Navigate Commit History

Checkout a Specific Commit

Temporarily switch to a previous commit state (read-only):

git checkout a1b2c3d

Replace "a1b2c3d" with your actual commit hash

Return to Latest Commit

Switch back to the most recent commit in your branch:

git checkout main

Replace "main" with your branch name if different

Force Checkout (Discard Changes)

Overwrite all local changes and return to the latest commit (WARNING: Permanent data loss):

git checkout -f main

Create Branch from Old Commit

Start a new branch from a historical commit (safe for modifications):

git checkout -b new-branch-name a1b2c3d

Find Commit Hashes

View abbreviated commit history with hashes:

git log --oneline

Branch Operations

Create New Branch

Start a new branch from your current commit:

git branch new-feature

Switch to Branch

Move to an existing branch:

git checkout branch-name

Create + Switch Branch

Combine branch creation and switching in one command:

git checkout -b hotfix-branch

List All Branches

View local branches (* marks current branch):

git branch

Delete Branch

Remove a merged branch (safe deletion):

git branch -d old-branch

Force Delete Branch

Delete unmerged branch (potential data loss):

git branch -D unmerged-branch

Merge Operations

Merge Branch

Combine another branch into your current branch:

git merge feature-branch

Fast-Forward Merge

Linear history merge (when possible):

git merge --ff-only main

No-Fast-Forward Merge

Always create merge commit (even when unnecessary):

git merge --no-ff dev

Abort Merge

Cancel a merge in progress with conflicts:

git merge --abort

View Merge Conflicts

List files with merge conflicts:

git status

Diff Operations

Unstaged Changes

View changes in working directory that haven't been staged:

git diff

Staged Changes

View changes that have been staged but not yet committed:

git diff --cached

All Changes (Staged + Unstaged)

View both staged and unstaged changes together:

git diff HEAD

Compare Two Branches

Show differences between two branches:

git diff main..dev

Compare Two Commits

Show changes between two specific commits:

git diff commit1 commit2

Check Changes Since Last Commit

See what changed since the last commit:

git diff HEAD
